# Excel Academy - School Website

A modern, responsive, and accessible website for Excel Academy built with HTML5, TailwindCSS, and vanilla JavaScript.

## Features

- **Fully Responsive** - Works on all device sizes from mobile to desktop
- **Accessible** - Follows WCAG 2.1 accessibility guidelines
- **Fast Performance** - Optimized assets and code for fast loading
- **Modern Design** - Clean, professional look with smooth animations
- **Easy to Maintain** - Well-structured codebase with clear documentation

## Getting Started

### Prerequisites

- Modern web browser (Chrome, Firefox, Safari, or Edge)
- Node.js (for development)
- npm or yarn (for development)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/school-website.git
   cd school-website
   ```

2. Install dependencies (for development):
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## Building for Production

To create a production build:

```bash
npm run build
```

This will optimize all assets and create a `dist` directory with the production-ready files.

## Testing

Run the test suite to ensure everything is working correctly:

1. Open `test-navigation.html` in your browser
2. Click "Run All Tests" to verify all pages and functionality
